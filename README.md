# Библиотека для работы с дисплеем Winstar WO2004B-TFH#

**** ВНИМАНИЕ! Использовать только с микроконтроллерами имеющими питание и сигналы напряжением 3.3 В (STM32, ESP32, ...)

#define RST_PIN PC3 // stm32f4 - выбор пина микроконтроллера для сигнала Reset
### LCD_wo2004b lcd(RST_PIN); - объявление объекта класса LCD_wo2004b 

Примеры работы с объявленным объектом:
### lcd.init(); - начальная инициализация дисплея
### lcd.rotateOn(); - поворот дисплея на 180 градусов
### lcd.setCursor(3, 0); - установка курсора по координатам (№ строки, № позиции в строке)
### lcd.print("Hello"); - вывод на экран текстовой строки с текущего положения курсора
### lcd.print("Display", 1, 6); - вывод на экран текстовой строки с указанием местоположения курсора ("текст", № строки, № позиции в строке)
### lcd.shiftStartLine(8); - установка количества линий смещения выводимого текста (и в положительную и в отрицательную сторону)
